# crystal-rpm

The [RPM] bindings for [Crystal] language based on [ruby-rpm-ffi].

It supports RPM 4.8.0 or later.

## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  rpm:
    github: lugia-kun/crystal-rpm
```
2. Run `shards install`
